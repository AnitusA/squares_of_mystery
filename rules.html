Square of Mysteries

Rules & Regulations

Follow these rules to keep the game fair, clear, and easy to run.

Game Rules

  • Teams move only when the admin enters a valid dice result.
  • Dare, treasure, and hex tiles can earn points when completed.
  • mine tiles act as minus tiles and move the team back by the penalty amount.
  • The hall display updates live with the latest tile event.
  • The first three teams to reach the final tile can be recorded as winners.

Regulations

  • The admin panel controls scoring, turn order, and board reset.
  • Do not edit stored game data unless you are the admin.
  • Use the hall screen only for viewing live progress and winners.
  • When a mine penalty is applied, both points and position must update together.
  • Keep the game fair by applying the same rules to every team.
